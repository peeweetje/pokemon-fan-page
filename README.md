# Pokemon-fan-page

Create a pokemon fan page for the purpose of making cool animations.

## What can you do?

- Look at the pokedex => limit = 100
- play a memory game.
- View evolution groups.
- Battle against a pokemon!
- Play a Quiz, are you a true pokemon master???
- Look at the pokemon guides for some tips and tricks!

## Features

### Pokedex
Browse through a collection of 100 pokemon with detailed information including abilities, moves, stats, and physical attributes. Each pokemon card features smooth animations and interactive elements.

### Memory Game
Test your memory with a fun pokemon-themed card matching game. Features include:
- Customizable difficulty levels
- Sound effects and animations
- Score tracking and high score persistence
- Confetti celebration for winning

### Evolution Groups
Explore pokemon evolution chains and learn about different evolution types including:
- Basic evolutions
- Trade evolutions
- Friendship evolutions
- Level-based evolutions

### Battle Simulator
Challenge yourself in turn-based battles against random pokemon. Features include:
- Health tracking and damage calculations
- Battle result summaries
- Win/loss statistics

### Pokemon Quiz
Test your pokemon knowledge with multiple-choice questions covering various aspects of the pokemon universe.

### Pokemon Guides
Learn tips and tricks about pokemon with animated guides featuring floating particles and interactive elements.

## Testing

This project uses Vitest for component testing. The test suite includes:
- Component rendering tests
- User interaction tests
- Mock data for API calls
- Comprehensive test coverage

Run tests with:
```bash
pnpm test
```

## Component Development

This project uses Storybook for component development and documentation. Storybook provides an isolated environment to develop, test, and document UI components.

Start Storybook with:
```bash
pnpm run storybook
```

Build Storybook for production:
```bash
pnpm run build-storybook
```
